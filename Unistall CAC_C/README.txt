\\Install CAC-C

Place both run.cmd and Install.ps1 into the extracted CAC-C folder, main directory, and run the bat as admin, you will asked to specify a path to move the folders too. Copy the path to the mods folder within the BattleTech directory. Make sure to remove quotation marks when copying the path.    

"The script simply will move and overwrite any and files that are in the folder"

\\Unistall CAC-C

Place Run.cmd & REMOVE_CAC_C.PS1 in the mods folder an execute the script it will remove all nessessary files.
